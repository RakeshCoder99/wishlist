# **App Name**: Reel Dreams

## Core Features:

- Add Movie: Allow users to add movie titles to their wishlist.
- Mark as Watched: Enable users to mark movies as watched.
- View Wishlist: Display the complete movie wishlist.
- Delete Entry: Allow users to delete movies from their wishlist.
- Intelligent Recommendations: Leverage AI to recommend movies to add to your wishlist based on watch history or genre preferences.

## Style Guidelines:

- Primary color: Deep cinematic purple (#4A235A) to evoke a sense of mystery and sophistication.
- Background color: A very dark desaturated purple (#1E1726).
- Accent color: Gold (#D4AC0D) for highlights and interactive elements to bring a touch of elegance.
- Headline font: 'Playfair' serif for elegance. Body font: 'PT Sans' sans-serif to pair with 'Playfair' for body text, offering readability and a modern feel.
- Use elegant, minimalist icons for actions like adding, deleting, and marking movies as watched.
- A clean, intuitive layout that allows easy navigation and focuses user attention on the movie wishlist.
- Subtle animations, like fade-in effects when adding movies, to enhance user experience.
